XSS简介

xss基本概念

xss原理

xss分类

  • 反射型

    把用户输入的数据“反射”给浏览器，需要用户点击链接才可以攻击成功，反射型xss也叫做非持久性xss。

  • 存储型
    >
    存储型xss会把用户输入的数据“存储”在服务器端。也叫持久性xss
  • DOM Based XSS
    >
    是通过修改页面的DOM节点形成xss，称之为DOM based XSS，从效果上来说也是反射型xss。

xss危害

  • Cooike劫持，盗取目标管理员COOKIE信息
  • 伪造钓鱼页面，诱骗行点击
  • 模拟用户身份执行操作

xss测试

xss防御

为了更好的设计xss防御方案，需要认清xss产生的本质原因。
本质是一种“HTML注入”，用户的数据被当成了HTML代码一部分来执行，从而混淆了原本的语义，产生了新的语义。
